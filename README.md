## Features

### ⭐️ User
- 회원가입
- 로그인/로그아웃
- 회원 정보 등에 대한 유효성 체크

### ⭐️ Card
- 카드 발급/충전/삭제
- 결제 기능

### ⭐️ Kiosk
- 메뉴 조회/생성
- 할인 적용
- 주문 기능

### ⭐️ Basket
- 메뉴 장바구니에 담기 기능
- 주문 개수 수정 및 메뉴 삭제 구현
